Employee Performance Calculation

From here, you can calculate a performance value for an employee in your company. This performance is based on leave count, number of rejected tasks and number of late submitted tasks.

A simple warning...The system uses monthly data and range of each parameter is between 0 and 10. If the values are differes from this range, the system will not work properly.
